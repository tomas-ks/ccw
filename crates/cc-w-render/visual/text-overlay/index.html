Text Overlay Visual Fixture

SDF glyphs from cc-w-render text layout

This page previews the Rust text overlay fixture. It is meant for quick visual QA of glyph size, edge crispness, outline behavior, and atlas packing. The PNG is generated from the same font layout and SDF atlas used by the renderer.

Regenerate the PNG from the repo root:

CC_W_TEXT_OVERLAY_VISUAL_OUT=crates/cc-w-render/visual/text-overlay/text-overlay.png \
cargo test -p cc-w-render dump_text_glyph_visual_fixture -- --ignored

Then open this index.html file directly in a browser. The page sizes the PNG by devicePixelRatio, so one image pixel maps to one physical screen pixel. Keep browser zoom at 100% for pixel-level QA.
